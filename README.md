# Artem Prokhorov GitHub Pages site

This is a Jekyll-based personal academic website prepared for GitHub Pages.

## How to use

1. Create a repository named `USERNAME.github.io` on your GitHub account.
2. Upload all files from this folder into the repository root.
3. Commit the files.
4. In GitHub, go to **Settings > Pages** and confirm deployment from the `main` branch.
5. Your website should appear at `https://USERNAME.github.io`.

## Files included

- `index.md` for the home page
- `research.md`
- `teaching.md`
- `red-book.md`
- `seminars.md`
- `consulting.md`
- `recruitment.md`
- `links.md`
- `_config.yml` for Jekyll settings
- `assets/css/style.scss` for light custom styling on top of the Minima theme

## Editing

Each page is plain Markdown, which makes updates much easier than hand-editing HTML. Replace placeholder entries with your own content and links.
